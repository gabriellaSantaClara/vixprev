- O sistema esta conectado com banco de dados Microsoft SQL Server;

- Um script SQL que cria o database, as tabelas usuarios e favoritos, e inseri o primeiro usuario tambem esta entre os arquivos;   

- A primeira tela Fomr1 é a tela de login onde um usuario ja esta no banco de dados como usuario "user" e senha "user";

- A segunda tela(frmprincipal) possui a possibildade de pesquisar um gif. Com a pesquisa é exibido a url dos primeiros gifs da pesquisa;

- Ainda na segunda tela existe a possibilidade de favoritar o gif pesquisado e ir para a tela de gerenciar gifs favoritos;

- Na terceira tela estao as opcoes mudar nome do gif favorito e deletar gif favorito;

- Para realizar as acoes anteriores é preciso pesquisar o gif na lista de favoritos.
